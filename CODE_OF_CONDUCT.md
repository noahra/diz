# Code of Conduct

## Our Standards

Be kind. Be constructive. Assume good intent.

Specifically:
- Welcoming and inclusive language
- Respectful disagreement — critique ideas, not people
- Accept feedback gracefully

Not acceptable:
- Harassment, insults, or personal attacks
- Trolling or deliberately disruptive behavior
- Publishing others' private information without consent

## Scope

This applies in all project spaces — GitHub issues, pull requests, and discussions.

## Enforcement

Violations can be reported by opening an issue or contacting the maintainer directly.
Responses will be proportional and fair.
